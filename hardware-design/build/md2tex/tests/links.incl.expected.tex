
if we put a link in the middle of a sentence, like \url{http://google.com} then we expect it to be linked

this should work equally for \url{https://yahoo.com}

\url{http://facebook.com} should work too, though we do not generally put links at the start of sentences

Multiple links in the same sentence is also allowed: \url{http://reddit.com} \url{https://nytimes.com} and also \url{http://instagram.com} for example
