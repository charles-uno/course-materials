1. Make a truth table for the boolean function $f(A,B,C) = (A\textrm{ xor }B)\textrm{ and }(\textrm{not }C)$.
2. Draw a logic circuit for the function $f(A,B,C)$ from above.
3. Draw a logic circuit which triples a two-bit unsigned binary integer. There should be two input bits ($A$ and $B$) and four output bits ($C$, $D$, $E$, and $F$) such that $0\mathrm{b}AB \times 0\mathrm{b}11 = 0\mathrm{b}CDEF$. 
    For example, $A=0$ and $B=1$ should give $C=0$, $D=0$, $E=1$, and $F=1$.
4. Draw a logic circuit with two 1-bit inputs, $A$ and $B$. There should be one 1-bit output, $C$, which is what we get if we flip the bits of $A$ and $B$, and then multiply the flipped bits.
    For example, if $A=1$ and $B=1$, the flipped bits would both be $0$, and their product would be $0$.
